== Description ==

This is a simple WooCommerce payment gateway plugin for Moneyflowz.

== Installation ==

1. Upload the entire `moneyflowz-payment-gateway` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce -> Settings -> Payments and enable "My Payment Gateway".
